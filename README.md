# Sistema de Gestión de Inventarios 2

Sistema de escritorio desarrollado en Python con interfaz gráfica
para la administración de inventarios.

## Descripción

Aplicación con interfaz gráfica que permite gestionar productos
y controlar el stock de manera visual e intuitiva.

## Tecnologías utilizadas

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)

## Características

- Registro y gestión de productos
- Control de stock
- Interfaz gráfica de escritorio
- Fácil de usar sin conocimientos técnicos

## Requisitos

- Python instalado
- Librería `tkinter` 

## Cómo ejecutarlo

- Instala dependencias si las hay pip install -r requirements.txt
- Ejecuta el programa python main.py
- Si no se encuentra Python instalado, descargarlo
